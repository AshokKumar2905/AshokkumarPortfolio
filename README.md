## 📌 Overview

This is my personal developer portfolio website.

It includes:

- Hero section with typing animation  
- Smooth scroll and fade-in animations  
- Dynamic Skills section  
- Dynamic Projects section  
- Mobile-friendly responsive design  
- Clean and simple UI built using **HTML, CSS, and JavaScript**

The portfolio helps visitors quickly understand who I am, what skills I have, and what projects I’ve built.

## ✨ Features

### ⭐ Typing Effect in Hero Section

The text **"Hello, I'm Ashokkumar D"** appears with a typing animation to create a dynamic introduction.

### ⭐ Scroll Fade-In Animations

All sections fade in smoothly as you scroll, enhancing the user experience and making the site feel more interactive.

### ⭐ Dynamic Skills Section

Skills are rendered using JavaScript from an array, making it easy to update or add new skills without changing the HTML structure.

### ⭐ Dynamic Projects Section

Projects are stored in JavaScript and displayed automatically on the page.  
Currently includes:

- Portfolio Website  
- Blog Platform  
- CSVServer Solution (recently added)

### ⭐ Mobile Menu Toggle

The navigation menu adapts to mobile screens with a toggle button, ensuring a good experience on smaller devices.
